# OfferCheck 💰

An AI-powered salary offer analyzer for tech roles. Paste your offer, get a market comparison, and receive a personalized negotiation script in seconds.

## Live Demo
[https://offercheck.onrender.com](https://offercheck.onrender.com)

## Features
- 📊 Market comparison against P25, Median, P75 benchmarks
- 🚨 Verdict: Lowballed / Fair / Above Market
- 🤖 AI-generated negotiation script (powered by Groq + LLaMA)
- 📍 Location-adjusted salary data
- 📱 Mobile-friendly dark UI

## Tech Stack
**Backend:** Java 17 + Spring Boot 4 + Groq API (LLaMA 3.1)

**Frontend:** React, mobile-first CSS

**Deployed on:** Render (free)
